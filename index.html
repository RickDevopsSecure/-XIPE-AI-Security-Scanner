Phase 2 Roadmap

Building the
custom model.

Every scan stores structured data in S3. After ~1,000 engagements, this trains a custom Llama 3 — zero external API dependency, zero per-scan cost.

01
Collect
Every scan saves attack prompts, AI responses, vulnerability labels, tech stack, severity. Sanitized — no credentials stored. Currently — records.
02
Train
At ~1,000 engagements, fine-tune Llama 3 or Mistral 7B on the XIPE dataset. Specialized for AI security — better signal-to-noise than any general model.
03
Deploy
The model runs on ECS alongside XIPE. Zero external API calls. Zero per-scan cost. Full control over the intelligence layer.
